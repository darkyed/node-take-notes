# node-take-notes

A web application build using **NodeJS**, where you can keep your favorite code snippets for different languages. It uses Mongoose ODM for **MongoDB** for the CRUD operations.



### Features
- Add a new language
- Add snippets for a given language
- Deleting snippets from a language
- Searching through snippets of a given language using tile of the snippet
